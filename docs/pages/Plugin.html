Kendal Plugin Manual


How to add plugin to IntelliJ IDEA?

  1. Click "File"
  2. Click "Settings"
  3. Select "Plugins" tab
  4. Click "Browse repositories..." button
  5. Search for "Kendal"
  6. Click "Install"
  7. Restart IntelliJ IDEA
